# 🚗 Sistema de Gerenciamento de Concessionária

Um sistema em Python para gerenciamento de veículos, permitindo cadastro, atualização, remoção e listagem de carros/motos diretamente pelo terminal.

---

## 🧠 Sobre o Projeto

Esse projeto simula uma **concessionária**, onde os veículos são armazenados em uma lista (`garagem`) contendo dicionários com informações como:

* Modelo
* Cor
* Quilometragem
* Ano
* Marca
* Placa

O sistema funciona em modo interativo via terminal.

---

## ⚙️ Como funciona

O sistema possui um menu com as seguintes opções:

```
1 - Cadastro de veiculo
2 - Atualização de veiculo
3 - Remoção de veiculo
4 - Listagem de veiculo
5 - Sair
```

O usuário escolhe uma opção e o sistema executa a ação correspondente.

---

## 🛡️ Validações

O sistema impede:

* Valores negativos para km e ano
* Veículos com ano ≤ 2015
* Campos vazios
* Atualizações inválidas

---

## 📚 Aprendizados

Esse projeto foi desenvolvido para praticar:

* Estruturas de repetição (`while`, `for`)
* Condicionais (`if`, `elif`, `else`)
* Listas e dicionários
* Manipulação de dados
* Entrada de dados com `input()`
